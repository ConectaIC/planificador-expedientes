Instrucciones rápidas:
1) En Supabase → Table Editor → SQL → pega y ejecuta 'supabase_schema.sql'.
2) Crea variables de entorno en tu app web (cuando la subamos a Vercel):
   - NEXT_PUBLIC_SUPABASE_URL
   - NEXT_PUBLIC_SUPABASE_ANON_KEY
   - SUPABASE_SERVICE_ROLE_KEY
3) Probamos inserción manual en Table Editor → expedientes.
